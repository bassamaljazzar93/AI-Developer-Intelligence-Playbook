Chapter 01

الفهم العام والتثبيت

Claude Code هو أداة تطوير تعمل داخل التيرمنال، وتتعامل مع المشروع كبيئة عمل وليس كمحادثة عادية.

Claude Chat vs Claude Code

العنصرClaude ChatClaude Code
المكانالمتصفح أو التطبيقالتيرمنال أو IDE
المشروعتنسخ أو ترفع ملفاتيعمل داخل مجلد المشروع
القوةشرح وتفكيرقراءة، تخطيط، تعديل، مراجعة

قبل التثبيت

  • جهز مشروعًا داخل Git.
  • اعرف أمر التشغيل.
  • اعرف أمر الاختبار أو طريقة الفحص اليدوي.
  • اكتب تعليمات مشروع مختصرة لاحقًا.

أوامر البداية

cd path/to/project
claude
أول طلب يجب أن يكون قراءة فقط: لا تبدأ بتعديل الملفات.
Read this project only. Do not edit files. Summarize the architecture, run command, test command, and risks.

علامات بداية صحيحة

جيد

يذكر ملفات فعلية، يشرح البنية، ولا يعدل إذا طلبت منه عدم التعديل.

خطر

يبدأ بتغييرات واسعة، يخترع أوامر، أو يدعي أنه اختبر دون تشغيل.
