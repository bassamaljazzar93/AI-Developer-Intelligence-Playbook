مكتبة Prompts جاهزة

هذه الصفحة للاستخدام اليومي. انسخ prompt وعدّله حسب المشروع.

قراءة المشروع

Read this project only. Do not edit files. Summarize project purpose, main folders, run command, test command, risks, and first safe improvement.

خطة قبل التنفيذ

Create a step-by-step plan. Do not edit files yet. For each step include files, risk level, verification method, and stopping point.

تنفيذ خطوة واحدة

Implement only step 1 from the plan. Keep changes minimal. Do not touch unrelated files. Summarize changed files and verification status.

مراجعة الفرق

Review the current diff as a strict reviewer. Do not edit. Focus on bugs, missing tests, risky assumptions, unclear naming, and unnecessary complexity.

تلخيص جلسة

Create a handoff summary:
- goal
- completed work
- files changed
- tests run
- risks
- exact next step
Prompt ذهبي: Read the relevant context first. Plan briefly. Make one minimal change. Verify if possible. Then summarize diff and next action.
