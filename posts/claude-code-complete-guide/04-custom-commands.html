الأوامر المخصصة

أي Prompt تكرره أكثر من مرة يجب أن يتحول إلى أمر مختصر قابل لإعادة الاستخدام.

متى تنشئ أمرًا؟

  • مراجعة diff.
  • توليد خطة اختبار.
  • تحديث documentation.
  • تلخيص الجلسة.
  • اقتراح commit message.

المكان

.claude/commands/

كل أمر يكون ملف Markdown داخل هذا المجلد.

مثال: أمر مراجعة

.claude/commands/review.md
Review the current git diff. Do not edit files.
Focus on bugs, missing tests, risky assumptions, and unnecessary complexity.
Return findings ordered by severity.
