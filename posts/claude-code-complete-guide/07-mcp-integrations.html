MCP

MCP والربط مع الأدوات

MCP يضيف سياقًا وأدوات خارجية لـ Claude Code، لكنه يجب أن يستخدم لحاجة واضحة وليس لتجميع إضافات بلا هدف.

أمثلة MCP مفيدة

GitHub

Issues، PRs، بحث في الكود، ومراجعات.

Filesystem

قراءة مصادر معرفة أو ملفات خارج مجلد المشروع.

Database

فهم schema وكتابة queries في بيئة تطوير.

Browser/Search

قراءة docs حديثة ومصادر مجتمع.

أسئلة قبل إضافة MCP

  1. هل يحل مشكلة متكررة؟
  2. هل يعطي سياقًا أفضل؟
  3. هل يمكن ضبط حدوده؟
  4. هل المصدر موثوق؟
  5. هل يتعامل مع بيانات حساسة؟
قاعدة: لا تضف MCP لأنه مشهور. أضفه عندما يكون لديك use case واضح.
