دليل المشاكل الميداني

معظم مشاكل Claude Code تأتي من نطاق غير واضح، غياب تحقق، أو جلسة طويلة جدًا.

مشاكل شائعة

يقرأ كثيرًا ولا ينجز

قلل النطاق واطلب سبب قراءة أي ملف إضافي.

يعدل ملفات غير مرتبطة

اطلب minimal change وراجع قائمة الملفات المتغيرة.

يعلن النجاح بلا اختبار

اطلب منه التمييز بين ما تم تشغيله وما لم يتم.

custom command لا يظهر

راجع المسار والاسم وأعد تشغيل Claude Code.

Prompt تشخيص سريع

Diagnose what went wrong in the last attempt. Do not edit. Identify scope, context, verification, and instruction issues.

أسئلة قبل اللوم على الأداة

  1. هل النطاق واضح؟
  2. هل طلبت خطة قبل التنفيذ؟
  3. هل يوجد اختبار؟
  4. هل تعليمات المشروع مكتوبة؟
  5. هل التغيير كبير جدًا؟
قاعدة: أصلح الـ workflow قبل أن تغيّر الأداة.
