Plugins, Skills, Agents

الهدف من الإضافات ليس التعقيد، بل تحويل الخبرة المتكررة إلى مكونات قابلة لإعادة الاستخدام.

الفرق بينهم

العنصرالفكرةأفضل استخدام
CommandPrompt محفوظمهمة متكررة
Skillمعرفة متخصصةنوع عمل محدد
Agentدور متخصصمراجع، معماري، كاتب اختبارات
Pluginحزمة متكاملةWorkflow لفريق أو مشروع

طريقة التدرج الصحيحة

Prompt متكرر → Command → Skill → Plugin

لا تبدأ ببناء plugin كبير. ابدأ بأمر واحد يحل مشكلة حقيقية.

أمثلة عملية

Code Review

Command + reviewer agent + checklist.

Frontend Design

Skill يثبت الهوية البصرية والتفاصيل.

القرار: إذا لم تستخدم prompt ثلاث مرات، لا تحوله إلى plugin.
