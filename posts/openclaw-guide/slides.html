التثبيت السريع

تحتاج Node.js 24+ فقط. لا توجد متطلبات إضافية.

تحقق من النسخة أولاً: node --version — يجب أن تكون 24.x
npm install -g openclaw@latest
openclaw onboard --install-daemon

الـ wizard سيسألك عن:

  • مزود النموذج (OpenAI، Anthropic، Ollama...)
  • قناة المراسلة (تيليغرام الأسرع)
  • تثبيت الـ daemon للتشغيل التلقائي
بعد الإعداد: افتح http://127.0.0.1:18789/ من متصفحك لواجهة التحكم الكاملة.

ChatGPT — استخدام اشتراكك الحالي

إذا عندك اشتراك ChatGPT Plus أو Codex، تقدر تشغّل OpenClaw عليه مباشرة بدون رسوم إضافية.

openclaw onboard --auth-choice openai-codex

# في بيئة headless (بدون متصفح)
openclaw onboard --auth-choice openai-codex --device-code
⚠️ الخطأ الأشهر: الربط ينجح لكن ما تضيف agentRuntime — فيشتغل على API key بدل الاشتراك.

أضف هذا في ~/.openclaw/openclaw.json:

{
  "plugins": {
    "entries": { "codex": { "enabled": true } }
  },
  "agents": {
    "defaults": {
      "model": { "primary": "openai/gpt-5.5" },
      "agentRuntime": { "id": "codex" }
    }
  }
}
تحقق: ارسل /usage full في المحادثة — إذا ما يحسب رسوم tokens فهو على الاشتراك.

OpenAI API Key

openclaw onboard --auth-choice openai-api-key
# أو مباشرة
openclaw onboard --openai-api-key "sk-..."

في openclaw.json:

{
  "env": { "OPENAI_API_KEY": "sk-..." },
  "agents": {
    "defaults": {
      "model": { "primary": "openai/gpt-5.4-mini" }
    }
  }
}
للتوفير: gpt-5.4-mini أرخص من gpt-5.5 وكافٍ للمهام اليومية.

نماذج محلية — Ollama (مجاني)

الخيار المجاني الكامل. يعمل offline. مثالي للتجربة الأولى.

# 1. ثبّت Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# 2. حمّل نموذج
ollama pull llama3.3

# 3. اربطه في openclaw.json

{
  "agents": {
    "defaults": {
      "model": { "primary": "ollama/llama3.3" }
    }
  }
}
إعداد القناة

تيليغرام — الأسرع والأسهل

  1. افتح @BotFather في تيليغرام
  2. أرسل /newbot واتبع التعليمات
  3. احتفظ بالـ token الذي يعطيك إياه
  4. في الـ terminal نفّذ: openclaw channels add telegram
  5. الصق الـ token عند الطلب
اختبار: ابحث عن اسم البوت في تيليغرام، ابدأ محادثة، وأرسل /status
إذا البوت ما ردّ: نفّذ openclaw doctor — يشخّص مشاكل الـ dmPolicy والاتصال تلقائياً.

نصائح ميدانية

نصائح ميدانية

ابدأ بمهام صغيرة

لا تطلب "أدر بريدي كله". ابدأ بـ "لخّص آخر 5 إيميلات". وسّع الصلاحيات تدريجياً.

الـ Daemon ضروري

بدونه يتوقف الـ gateway لما تغلق الـ terminal. --install-daemon يحل المشكلة نهائياً.

/compact يوفّر tokens

لما الجلسة تطول، /compact يضغط السياق ويبقي الأهم — بدون تأثير على الجودة.

Ollama للتجربة

قبل ما تدفع لأي مزود، جرّب Ollama مجاناً. إذا الـ workflow مفيد، انتقل لنموذج أقوى.

واجهة الويب: http://127.0.0.1:18789/ — لوحة تحكم كاملة للجلسات والإعدادات والـ logs. أفضل من تعديل JSON يدوياً.

القرار النهائي

هل تستخدمه؟

✓ استخدمه إذا

  • عندك اشتراك ChatGPT أو Claude
  • تبي تتحكم من هاتفك عبر واتساب/تيليغرام
  • ما تمانع تعطي أداة محلية صلاحية على الملفات
  • تبي تجرب agentic AI بدون كود

✗ تجنّبه إذا

  • ما عندك مزود AI واضح
  • ما تريد أداة بصلاحيات واسعة على جهازك
  • تبحث عن أداة enterprise بدعم رسمي
القاعدة: ابدأ بـ Ollama مجاناً. إذا الـ workflow أثبت قيمته في 24 ساعة، انتقل لنموذج أقوى.
