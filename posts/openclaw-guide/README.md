# OpenClaw — الدليل التشغيلي الكامل

OpenClaw هو assistant شخصي مفتوح المصدر يعمل **محلياً على جهازك** ويتحكم فيه عبر واتساب أو تيليغرام أو أي تطبيق مراسلة تستخدمه.

يتصل بملفاتك، بريدك، كالندرك، ويتحرك في المتصفح — كلها من محادثة واحدة.

---

## إجابات الأسئلة السبعة

### 1. إيش يشتغل اليوم فعلاً؟
- إرسال إيميلات وإدارة البريد الوارد
- إضافة أحداث في Google Calendar وحذفها
- تشغيل أوامر shell وتنفيذ سكريبتات
- التصفح في المواقع وملء النماذج
- تذكيرات ومهام تلقائية في الخلفية (cron jobs)
- ذاكرة دائمة تتراكم مع الوقت

### 2. إيش أسرع إعداد؟
```
npm install -g openclaw@latest
openclaw onboard
```
تلغرام هو الأسرع للإعداد الأول.

### 3. إيش مجاني؟
- OpenClaw نفسه: **مجاني ومفتوح المصدر**
- النموذج: تدفع للمزود (OpenAI API) أو تستخدم اشتراكك الحالي في ChatGPT أو Claude
- Ollama (نماذج محلية): **مجاني بالكامل**

### 4. إيش المشاكل الشائعة؟
- الحاجة لـ Node 24 (نسخ أقدم تتعطل)
- خطأ في الـ dmPolicy عند أول رسالة من واتساب
- نسيان تفعيل Codex runtime عند استخدام اشتراك ChatGPT
- انقطاع القناة بعد إعادة تشغيل الجهاز إذا لم يُشغّل الـ daemon

### 5. إيش النماذج المقترحة؟
| الاستخدام | النموذج المقترح |
|-----------|----------------|
| ChatGPT اشتراك موجود | `openai/gpt-5.5` + Codex runtime |
| OpenAI API key | `openai/gpt-5.4-mini` (اقتصادي) |
| Claude | `anthropic/claude-opus-4-6` |
| مجاني محلي | Ollama + `llama3.3` |
| سريع ورخيص | `openai/gpt-5.4-mini` أو `deepseek/deepseek-v4-flash` |

### 6. إيش الأوامر المطلوبة؟
راجع [commands.md](commands.md) للقائمة الكاملة.

### 7. إيش تختبره خلال 24 ساعة؟
1. **الساعة 1:** ثبّت، شغّل، وأرسل أول رسالة عبر تيليغرام
2. **الساعة 2:** اطلب منه يقرأ ملف محلي ويلخّصه
3. **الساعة 4:** اطلب منه يرسل إيميل اختباري
4. **الساعة 8:** اطلب منه يضيف حدث في الكالندر
5. **الساعة 24:** قرار: هل يوفّر وقتاً حقيقياً؟

## المتطلبات

- **Node.js:** إصدار 24 (مُوصى به) أو 22.14+ كحد أدنى
- **npm أو pnpm**
- **مزود AI:** API key أو اشتراك ChatGPT/Claude
- **قناة مراسلة:** تيليغرام (الأسرع)، واتساب، سلاك، ديسكورد، أو غيرها

---

## البنية الداخلية

```
~/.openclaw/
├── openclaw.json          ← ملف الإعداد الرئيسي
├── workspace/             ← مجلد عمل الـ agent الافتراضي
└── agents/
    └── <agentId>/
        └── agent/
            └── auth-profiles.json   ← بيانات المصادقة
```

واجهة الويب المحلية: `http://127.0.0.1:18789/`

---

## القرار

**استخدمه إذا:** عندك اشتراك ChatGPT أو Claude وتبي تتحكم فيه من الهاتف عبر واتساب أو تيليغرام.

**تجنّبه إذا:** ما عندك مزود AI واضح أو ما تريد تعطي أداة محلية صلاحية على بريدك وملفاتك.

**اختبره أولاً إذا:** كنت مهتم بالـ agentic AI لكن ما جربت نماذج محلية من قبل — ابدأ بـ Ollama مجاناً.
