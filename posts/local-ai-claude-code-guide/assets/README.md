# Assets

This folder is reserved for slide images and visual assets.

Recommended naming:

```text
slide-01-cover.png
slide-02-how-it-works.png
slide-03-ollama-fast-start.png
slide-04-manual-setup.png
slide-05-options-comparison.png
slide-06-context-length.png
slide-07-troubleshooting.png
slide-08-model-selection.png
slide-09-execution-plan.png
```

If binary image upload is not available through the current tool, export the images locally and push them with Git:

```bash
git add posts/local-ai-claude-code-guide/assets/*.png
git commit -m "Add Local AI Claude Code slide images"
git push
```
