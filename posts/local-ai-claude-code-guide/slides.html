1

2

install from ollama.com

3

ollama pull qwen3.5
4

شغّل

ollama launch claude --model qwen3.5
Manual Setup

الربط اليدوي مع Claude Code

ANTHROPIC_AUTH_TOKEN=ollama
ANTHROPIC_API_KEY=
ANTHROPIC_BASE_URL=http://localhost:11434

claude --model qwen3.5
تنبيه مهم

لا تضف /v1 في عنوان Ollama. العنوان الصحيح هو http://localhost:11434. خطأ 404 غالبًا سببه هذا التفصيل.

Performance

الأداء والسياق Context Length

  • للمهام Agentic Coding: يفضل 64K إن توفر العتاد.
  • على CPU أو جهاز ضعيف: ابدأ بـ 16K أو 32K.
  • السياق الكبير يحتاج VRAM / RAM كافية.
  • إذا زاد الحمل على CPU ستصبح التجربة بطيئة جدًا.
OLLAMA_CONTEXT_LENGTH=64000 ollama serve
