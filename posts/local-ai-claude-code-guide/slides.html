داخل Claude Code

دليل عملي بأسلوب Claude الهادئ: خطوات واضحة، تقدم محفوظ، أوامر قابلة للنسخ، وتحذيرات عملية. الهدف قرار خلال 24 ساعة وليس بحث بلا نهاية.

ابدأ الدليلالصفحة الرئيسيةملف الأوامرالدليل التشغيلي

نفس هوية الصفحة الرئيسية: خلفية دافئة، نص داكن، ولمسات clay/orange مريحة للعين.

1

ما الهدف؟

تقليل التكلفة والتجريب السريع، لا استبدال Claude بالكامل.

يفيدك في

شرح ملفات، Refactor صغير، اختبارات، وتجارب كثيرة.

لا يكفي في

قرارات معمارية كبيرة أو مراجعة أمنية عميقة.

قاعدة: اختبر مهمة حقيقية ثم قرر. لا تنخدع باسم الموديل.
2

كيف تعمل المعمارية؟

Claude Code يتصل بـ API. الموديل يعمل في Backend.

Claude Code

واجهة Agentic Coding.

Backend

تبسيط: Claude Code هو السائق، الموديل المحلي هو المحرك، والـ API هو ناقل الحركة.
3

أسرع تشغيل: Ollama

ابدأ بهذا قبل أي تعقيد.

أوامر التثبيت الكاملة في commands.md. بعد التثبيت استخدم:

ollama pull qwen3.5
ollama launch claude --model qwen3.5
اختبار: اطلب فهم مشروع واحد فقط بدون تعديل ملفات.
Inspect this project. Do not edit anything. Give me a 5-point refactor plan only.

الربط اليدوي

$env:ANTHROPIC_AUTH_TOKEN="ollama"

Context Length

السياق الصغير يضعف الفهم، والكبير بلا عتاد يبطئك.

CPU فقط

ابدأ بـ 16K أو 32K.

GPU جيد

جرّب 64K تدريجيًا.

OLLAMA_CONTEXT_LENGTH=32000 ollama serve
OLLAMA_CONTEXT_LENGTH=64000 ollama serve
