## هيكل كل دليل

كل دليل يجيب على:

1. ما الذي يعمل اليوم؟
2. ما أسرع طريقة للبدء؟
3. ما المجاني أو منخفض التكلفة؟
4. ما المشاكل الشائعة وحلولها؟
5. ما النماذج أو الأدوات الموصى بها؟
6. ما الأوامر المطلوبة؟
7. ما الذي يجب اختباره خلال 24 ساعة؟
